# Bookin Frontend AGENTS.md

## Purpose
Owns the BookIn React frontend application workspace and state.

## Ownership
Platform Frontend Engineering

## Local Contracts
- All UI components, state management, and asset files live in `src/`.
- Communication with the backend is done via relative/absolute API routes, configured in Vite.

## Work Guidance
- Use `npm run dev` to run the development server locally.
- Build production assets using `npm run build`.

## Verification
- Run `npm run build` or use the linter to verify code correctness and compatibility.
